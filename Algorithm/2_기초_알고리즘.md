# \# 기초 알고리즘

 

- 이분 탐색(Binary Search)

  => 탐색 범위를 <u>반으로 좁혀감</u>으로써 탐색하는 방법

  - 순서

    1. 정렬이 필수적

    2. left, right, mid를 설정

       => left ~ right 가 탐색 영역

       ​      mid는 left, right 중간 지점

    3. mid와 찾고자 하는 값과 비교

       ㄱ. **구할 값이 mid 보다 크면** 

       ​		다음 탐색 영역 => mid+1 ~ right

       ㄴ. **구할 값이 mid 보다 작으면** 

       ​		다음 탐색 영역 => left ~ left-1

    4. left > right 가 될 때까지 또는 찾고자 하는 값을 찾을 때까지 반복

       => left > right 가 되면 값을 찾지 못한 것





- Hash Table 구현
