## 힙(Heap)



- <u>우선순위 큐를 위해 만들어진 자료구조</u>



##### *우선순위 큐?*

- 데이터들에 우선순위를 부여

  => 우선순위가 높은 데이터부터 먼저 나감



- 힙의 용례
  - 시뮬레이션 시스템
  - 작업 스케줄링
  - 수치해석 계산
  - 등..
- 삽입 / 삭제 => O(logn)





#### \*\* 힙

- 완전 이진 트리의 일종

- 최대값, 최소값을 빠르게 찾기 위한 자료구조

  => 반정렬 상태,,

- 힙 트리는 중복값 허용, 이진 탐색 트리는 중복값 비허용



##### \* 힙의 종류

1. Max Heap

   > 부모 노드의 키 값이 자식 노드의 키 값 보다 크거나 같은 완전 이진 트리

2. Min Heap

   > 부모 노드의 키 값이 자식 노드의 키 값 보다 작거나 같은 완전 이진 트리



##### 구현 사항

- 보통 배열로 구현

  => 보통 구현의 용이성을 위해 첫 번째 인덱스를 사용하지 않고,

  ​	1부터 사용

- 왼쪽 자식 index = (부모 index) * 2

  오른쪽 자식 index = (부모 index) * 2 + 1

  부모 index = (자식 index) / 2

- 요소 삽입 시

  1. 우선 맨 끝 노드로 삽입
  2. 부모 노드와 비교하며 swap 규칙 만족 시까지 올라감

- 요소 삭제 시

  1. 루트 노드만 삭제
  2. 맨 끝 노드를 루트 노드 위치로 가져옴
  3. 루트 노드를 아래로 내릴 수 있을 때까지 swap => **<u>힙의 재구성</u>**
